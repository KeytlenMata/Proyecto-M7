# Proyecto Módulo 7 – Calculadora Básica en Python – Trabajo Individual

## Descripción

Programa en Python que implementa una **calculadora básica** con interfaz de consola amigable. Permite realizar las cuatro operaciones aritméticas fundamentales: suma, resta, multiplicación y división, utilizando un diseño modular y buenas prácticas de programación.

El sistema solicita dos números al usuario y le permite elegir la operación deseada mediante un menú numerado. Incluye manejo de errores para divisiones entre cero, validación de entradas numéricas y retroalimentación clara en cada paso. Tras mostrar el resultado, se despide del usuario con un mensaje de cierre.

Este proyecto fue desarrollado de forma **individual** por **Keytlen Mata** como parte de la actividad del **Módulo 7**.

## Características

- **Operaciones matemáticas básicas**:  
  - Suma  
  - Resta  
  - Multiplicación  
  - División (con protección contra división por cero)  
- **Manejo de errores**:  
  - Validación de entradas numéricas (`float`)  
  - Mensaje de error al dividir entre cero  
  - Reintento automático si se selecciona una opción inválida en el menú  
- **Interfaz amigable**:  
  - Mensajes claros con emojis  
  - Separadores visuales (`=` y `-`) para mejor legibilidad  
  - Bienvenida y despedida personalizadas
